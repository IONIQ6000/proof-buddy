How to use:
TFL Syntax Rules:
Atomic sentences can be represented with uppercase letters A-Z
FOL Syntax Rules:

The following rules explain how to represent an FOL expression:

Names (Objects): a-r (lowercase)

Variables: s-z (lowercase)

Predicates: A-R (uppercase)

Domains (Sets): S-Z (uppercase)

Premises:
Separate premises using semicolons (e.g. "A; B").
Entering symbols:
To enter math symbols, use the bold escape commands below while typing.

Disjunction \or ∨
Negation \not ¬

Contradiction \con ⊥
Universal Quantifier \all ∀
